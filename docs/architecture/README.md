# アーキテクチャドキュメント

このディレクトリには、Figma UI
Reviewerのシステムアーキテクチャに関するドキュメントが含まれます。

## 📚 ドキュメント一覧

### 計画中のドキュメント

- **overview.md** - システム全体の概要、技術スタック、コンポーネント構成
- **data-flow.md** - 詳細なデータフロー、シーケンス図、リクエスト→評価→レスポンスの流れ
- **agent-system.md** - 評価エージェントの設計原則、BaseEvaluationAgentの責務、拡張方法
- **type-system.md** - 共通型定義（shared/）の説明、型安全性の保証方法
- **security.md** - セキュリティ考慮事項（ReDoS対策、入力検証、APIキー管理）

## 🎯 このカテゴリの目的

システムの構造と設計原則を理解するための情報を提供します。新規開発者のオンボーディングや、システム全体の理解が必要な場合に参照してください。
