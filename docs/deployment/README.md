# デプロイメントドキュメント

このディレクトリには、ビルドとデプロイに関するドキュメントが含まれます。

## 📚 ドキュメント一覧

### 計画中のドキュメント

- **environment-variables.md** - 環境変数一覧と説明（ANTHROPIC_API_KEY、PORT、NODE_ENV、CORS_ORIGIN）
- **build-process.md** - ビルドプロセス詳細（TailwindCSS、TypeScriptコンパイル、バンドル）
- **production-deployment.md** - 本番環境へのデプロイ手順、環境設定
- **monitoring.md** - ログ管理、パフォーマンス監視、エラートラッキング

## 🎯 このカテゴリの目的

本番環境へのデプロイと運用管理のための情報を提供します。リリース作業や本番環境のトラブルシューティング時に参照してください。
