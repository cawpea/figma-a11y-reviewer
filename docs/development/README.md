# 開発ドキュメント

## 📚 ドキュメント一覧

### 計画中のドキュメント

- **getting-started.md** - 初回セットアップ手順（環境変数、依存関係インストール、ビルド）
- **development-workflow.md** - 日常的な開発フロー（watch モード、ホットリロード、デバッグ）
- **coding-standards.md** - コーディング規約、命名規則、TypeScript使用ガイドライン
